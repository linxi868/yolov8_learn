主题
  此为yolov8.n小模型的导出教学
  适合用来学习ONNX导出和Tensor RT推理引擎的编译
  后续会加入如何实现在端侧进行部署

  
文件内容介绍
  包含模型网络框架(nets)，模型权重(weight)，导出ONNX文件(yolov8_onnx)，测试模型帧率（test_torch_fps),引用文件(utils)


ONNX模型导出使用手册
  本文件引入命令行传参，可通过命令 python yolov8_onnx.py 参数名 参数值 参数名 参数值............回车即可自动执行文件        #可用python yolov8_onnx.py -h 查看参数类型

  本文件若想导出其他模型，只需添加新模型的模型框架和权重文件（需from import导入模型），传参时进行修改对应量   #仅用于视觉模型，大语言模型需使用Tensor RT-LMM进行配置
  并在模型注册表（MODEL_REGISTY）中添加模型名和模型的输入形状，输出类别等信息，即可调用


代码大纲
  1.添加logger进行迅速定位
  2.添加模型注册表，并对库的版本进行检查
  3.定义基础命令行传参
  4.检查导出环境
  5.查看GPU内存占用
  6.加载模型及其权重
  7.虚拟输入配置
  8.动态维度配置
  9.进行模型简化和ONNX模型导出
  10.精度对齐检验
  11.性能测试
  12.主函数


注意事项
  1.算子版本过低会导致模型混乱
  2.在6中可自定义关键层检验，防止模型导出缺失
  3.模型注册表添加模型框架时，注意模型参数定义的导入
  4.函数调用都在主函数中，但是否调用可在命令行传参时进行配置
  5.模型不一样，在虚拟输入配置时归一化参数改变
  6.性能测试功能不够完善，可根据需求添加，后续我会进行传入
  7.暂时不支持自定义算子的导出，主播还没掌握cuda
  8.如果动态维度未开，在Tensor RT编译时不可用
  9.如需记录每次错误点，可在1中设置handlers中设置mode为a,每次运行追加在下面
